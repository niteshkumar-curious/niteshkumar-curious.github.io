ESP32 OTA Firmware Update System

This project demonstrates an IoT-based remote device management system using ESP32, where firmware updates can be performed over-the-air (OTA) without physical access.

Problem Statement

Traditional embedded systems require physical connection for firmware updates. This is inefficient for deployed devices. This project enables secure remote updates.

System Architecture

  • ESP32 (ESP-IDF)
  • MQTT communication
  • HTTPS firmware download
  • GitHub firmware hosting

Features

  • OTA updates
  • Secure HTTPS download
  • MQTT control
  • FreeRTOS multitasking

Working

ESP32 connects to WiFi and listens to MQTT. On OTA command, it downloads firmware via HTTPS and updates itself, then restarts.

OTA Flow

  1. Connect to WiFi
  2. Receive OTA command
  3. Download firmware
  4. Update and restart

OTA Code


esp_http_client_config_t http_config = {
    .url = OTA_FIRMWARE_URL,
    .crt_bundle_attach = esp_crt_bundle_attach,
};

esp_https_ota_config_t ota_config = {
    .http_config = &http_config,
};

esp_https_ota(&ota_config);
